# Rick and Morty API

Aplicação desenvolvida em **React + Vite** para consumir a [Rick and Morty API](https://rickandmortyapi.com/) utilizando **Axios**.  
O projeto foi criado como exercício de aprendizado junto com meu professor, com foco em entender como integrar APIs em aplicações front-end.

---

## 🚀 Tecnologias utilizadas
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Axios](https://axios-http.com/)
- [JavaScript (ES6+)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)
- [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS)

## 🎨 Funcionalidades
- Exibição dos personagens da série **Rick and Morty**.
- Cada card mostra:
  - Nome
  - Espécie
  - Status (vivo, morto ou desconhecido)
  - Origem/dimensão
- Interface simples e organizada com componentes reutilizáveis.

## 📦 Instalação e uso

### 1. Clone o repositório:
`bash`
```
git clone https://github.com/BielVereda/Rick_and_Morty_API.git
```

### 2. Acesse a pasta do projeto:

`bash`
```
cd Rick_and_Morty_API
```

### 3. Instale as dependências:
`bash`
```
npm install
```

### 4. Execute o projeto:
`bash`
```
npm run dev
```

### 5. Abra no navegador:
```
http://localhost:5173
```
